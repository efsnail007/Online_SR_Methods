ОТЧЕТ ПО УЧЕБНОЙ ПРАКТИКЕ

Тема: разработка клиент-серверного приложения для повышения разрешения кадров видеопотока с веб-камеры в реальном времени

Содержание

Введение

1 Анализ задачи и обоснование выбора технологий
1.1 Постановка задачи практики
В подразделе планируется описать цель практики, исходное задание, основные требования к приложению и ожидаемый результат разработки.

1.2 Особенности обработки кадров видеопотока и повышения разрешения изображений
В подразделе будет рассмотрена специфика работы с отдельными кадрами видеопотока, а также кратко описаны подходы к увеличению разрешения изображений.

1.3 Обоснование выбора технологического стека
В подразделе будет объяснен выбор FastAPI для серверной части, React/Vite для клиентской части, OpenCV и PyTorch для обработки изображений, а также Docker Compose для запуска системы.

2 Проектирование и разработка клиент-серверного приложения
2.1 Общая архитектура системы
В подразделе будет приведена общая схема работы приложения: захват кадра на клиенте, отправка данных на сервер, обработка изображения и возврат результата пользователю.

2.2 Реализация серверной части
В подразделе будет описана структура backend-приложения, загрузка модели Real-ESRGAN, REST API для обработки изображений, поддержка режима bicubic и выдача служебных метаданных.

2.3 Реализация клиентской части
В подразделе будет рассмотрен интерфейс веб-приложения, работа с веб-камерой, захват кадров через canvas, отправка кадров на сервер и отображение обработанного результата.

2.4 Организация обмена данными между клиентом и сервером
В подразделе будет описан формат передачи кадров, параметры запросов к API, обработка ответов сервера и вывод характеристик обработки в интерфейсе.

2.5 Конфигурирование и запуск проекта
В подразделе будет показана структура конфигурации проекта, использование переменных окружения и запуск компонентов через Docker Compose.

3 Отладка, тестирование и руководство пользователя
3.1 Отладка приложения в реальных условиях
В подразделе будет описан процесс проверки работы приложения при захвате кадров с веб-камеры, типовые ошибки и способы их устранения.

3.2 Тестирование серверной части приложения
В подразделе будет рассмотрен набор автоматических тестов для API, кодирования и декодирования изображений, а также сервиса повышения разрешения.

3.3 Проверка работоспособности пользовательского сценария
В подразделе будет описан сценарий работы пользователя: запуск системы, включение камеры, выбор метода обработки, получение и визуальный анализ результата.

3.4 Руководство пользователя
В подразделе будет приведена краткая инструкция по запуску приложения и работе с веб-интерфейсом.

3.5 Ограничения текущей реализации и направления развития
В подразделе будут перечислены ограничения текущей версии приложения и возможные пути его дальнейшего развития.

Заключение
В заключении планируется подвести итоги выполненной практики, оценить достигнутые результаты и сформулировать практическую значимость разработанного приложения.
